Mark My Words

This website provides an online profile for storing notes in markdown format. Notes can be saved and viewed in rendered markdown, offering a clean and readable presentation. The interface mimics a file explorer, allowing the creation and management of files, folders, and subfolders.

Key Features

  • Templates

    Create note templates for common note structures, accessible from a drop-down menu during editing.

  • Downloading and Uploading

    Download notes in markdown or HTML format, or upload markdown files. Whole directories can be downloaded as ZIP files.

  • Sharing and Access Control

    Configure view and edit permissions for other accounts via a note's settings page, with link sharing available.

  • Public Directory

    Each user has a public directory visible to others via a shareable profile link.

  • Images

    Embed images in notes using markdown syntax, manage images in user settings.
